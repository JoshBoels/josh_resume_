Project 2: Home Reminder System for Dementia Patients

Details about Project 2.

Project Description

This project was a group project with me and one other person. We made a home reminder system for dementia patients. A nurse or family member is able to put in reminders through the website that I made and then those reminder will show up on the device to remind the patient of daily tasks such as taking pills.

Technologies Used

  • VS Code, GitHub
  • Breadboards, wires, LCD, push buttons, LEDs, NPN transistors
